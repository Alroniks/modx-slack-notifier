This file shows the changes in recent releases of Slackify.

Slackify 0.4.5-alpha (September 18, 2015)
========================================
- Fixed minor bugs with configuration of message class

Slackify 0.4.4-alpha (September 18, 2015)
========================================
- Added plugins config to builder

Slackify 0.3.0-alpha (September 18, 2015)
========================================
- Added new system settings to package
- Added setup options resolver for fast setup notifier
- Added lexicons for systems settings: English, Russian and Belarusian
- Added support of new options in Notifier class

Slackify 0.2.0-alpha (September 17, 2015)
========================================
- Added Namespace generator to build script

Slackify 0.1.5-alpha (September 17, 2015)
========================================
- Created package builder
- Created classes for Slack messages entities
