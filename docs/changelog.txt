This file shows the changes in recent releases of Slackify.

Slackify 0.8.0-pl (September 24, 2015)
========================================
- Added plugin ms2OrderStatusSlackify for send notifies when order status changed

Slackify 0.7.0-beta (September 24, 2015)
========================================
- Added plugin NewTicketTicketsSlackify for send notifies when new ticket created

Slackify 0.6.2-beta (September 24, 2015)
========================================
- Added class Link for generates valid links for Slack
- Added plugin NewCommentTicketsSlackify for send notifies when new comments added

Slackify 0.5.0-beta (September 23, 2015)
========================================
- Added plugin for notify to Slack when 404 error happens

Slackify 0.4.10-alpha (September 23, 2015)
========================================
- Improved Message class for support icon and channel override

Slackify 0.4.9-alpha (September 22, 2015)
========================================
- Added missed extension resolver

Slackify 0.4.8-alpha (September 22, 2015)
========================================
- Fixed issue with setup resolver. Settings for entry point and channel not saved during setup process.

Slackify 0.4.6-alpha (September 18, 2015)
========================================
- Fixed minor bug with sender setting

Slackify 0.4.5-alpha (September 18, 2015)
========================================
- Fixed minor bugs with configuration of message class

Slackify 0.4.4-alpha (September 18, 2015)
========================================
- Added plugins config to builder

Slackify 0.3.0-alpha (September 18, 2015)
========================================
- Added new system settings to package
- Added setup options resolver for fast setup notifier
- Added lexicons for systems settings: English, Russian and Belarusian
- Added support of new options in Notifier class

Slackify 0.2.0-alpha (September 17, 2015)
========================================
- Added Namespace generator to build script

Slackify 0.1.5-alpha (September 17, 2015)
========================================
- Created package builder
- Created classes for Slack messages entities
